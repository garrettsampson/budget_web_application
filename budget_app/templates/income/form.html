Income Calculation

Enter your weekly income details below. We will calculate your gross and net pay for this week and store it under the selected year and month.

{# ---------------------------------------------------------------------------------- THE FORM ---------------------------------------------------------------------------------- - method="post" means a POST request is sent to the same URL (/income). - Flask receives data through request.form. - `class="bg-white shadow ..."` uses Tailwind utility classes. #}
{# ========================================================================== YEAR INPUT ========================================================================== • The name="year" MUST match what app.py expects: year = int(request.form["year"]) • The value is prefilled from app.py using variables: current_year = today.year #}
{# ========================================================================== MONTH INPUT ========================================================================== • Months must be 1–12. • Pre-filled using current_month from app.py. #}
{# ========================================================================== WEEK INDEX ========================================================================== • This is NOT tied to actual calendar dates. • It’s the user’s own categorization (Week 1, Week 2, etc.). • Stored under IncomeWeek.week_index. #}

{# ========================================================================== HOURLY PAY INPUT ========================================================================== • The user enters a decimal number (example: $13.25/hr). • step="0.01" allows cent-level precision. • name="hourly_pay" matches the backend: hourly = float(request.form["hourly_pay"]) #}
{# ========================================================================== HOURS WORKED ========================================================================== • Again, step="0.01" allows: - fractional hours (e.g., 37.5) - decimal precision • Tied to request.form["hours_worked"] #}
