When changing model.py:
- do not delete the DB
- instead:
    - run (in terminal)
        - flask db migrate -m "(Describe what changed)"
    - then run (in terminal)
        - flask db upgrade

- if you get an error:
    - Error A "Could not locate a Flask application"
        - open terminal and run
            - set FLASK_APP=app.py
            - set FLASK_ENV=development
    - Error B "No module named flask_migrate"
        - activate venv
        - open terminal and run 
            - pip install Flask-Migrate


When going forward with changing models.py, do this:

    .\venv\Scripts\python.exe -m flask db migrate -m "describe change"
    .\venv\Scripts\python.exe -m flask db upgrade
    git add .
    git commit -m "Describe database change"
    git push
